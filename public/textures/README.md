# Earth Textures

This directory should contain the following texture files for the Enhanced Earth component:

## Required Files:

1. **planet_color.jpg** - Earth day texture (color map)
2. **planet_normal.jpg** - Normal/bump map for surface detail
3. **planet_rough.jpg** - Roughness map for material properties
4. **planet_night.jpg** - Night lights texture (cities at night)
5. **planet_clouds.jpg** - Cloud layer texture

## Recommended Sources:

- NASA Blue Marble: https://visibleearth.nasa.gov/collection/1484/blue-marble
- Solar System Scope Textures: https://www.solarsystemscope.com/textures/
- Free Earth Textures: http://planetpixelemporium.com/earth.html

## Format Requirements:

- Format: JPEG or PNG
- Recommended resolution: 4K or 8K for best quality
- Files should be equirectangular projection (360° x 180°)

## Fallback:

If textures are not provided, the Earth component will use solid blue/white colors as a fallback.
